### High-level Options

* **`height` / `width`**; Defines explicit bar dimensions when required.
* **`spacing`**; Sets horizontal spacing between modules.
* **`margin` / `padding`**; Adjusts bar-level layout geometry.
* **`modules-left` / `modules-center` / `modules-right`**; Declares module placement structure.
* **`name`**; Assigns an identifier when running multiple bars simultaneously.
* **`reload_style_on_change`**; Enables dynamic CSS reloading.
* **`gtk-layer-shell` toggles**(e.g., `passthrough`); Advanced integration controls for compositors.
